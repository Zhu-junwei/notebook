# 设置

## 自动保存

File > Auto Save

## 字体大小

设置 > 工作区 > 文本编辑器 > 字体 > Font Size

## 设置/查看快捷键

- 设置 > 键盘快捷方式

- Ctrl + K , Ctrl + S



# 快捷键

快捷键参考：https://code.visualstudio.com/docs/reference/default-keybindings

```
Ctrl + Shift + p  # 打开控制面板
Ctrl + `  # 开关terminal
Ctrl + Shift + Y  # 打开调试控制台
Ctrl + p  # 转到文件
Ctrl + Shift + E  # 打开资源管理器
Ctrl + Shift + F  # 在文件中查找
Ctrl + Shift + G  # 源代码管理
Ctrl + Shift + D  # 打开调试控制台
Ctrl + Shift + X  # 打开扩展
Ctrl + Alt + i  # 打开AI聊天
Alt + Shift + f # 格式化代码
Ctrl =/-  # 调整字体大小
Ctrl + ,  # 打开设置
```
**编辑**

```
Ctrl + X  # 剪切当前行
Ctrl + C  # 复制当前行
Ctrl + V  # 粘贴
Ctrl + D  # 选中单词
Ctrl + <- / ->  # 左右选中
Ctrl + Shift + k # 删除当前行
Ctrl + Enter  # 下方插入行
Ctrl + Shift + Enter  # 上方插入行
Alt + Down  # 下移
Alt + Up   # 上移
Shift+Alt+Down   # 下复制
Shift+Alt+Up   # 上复制
Ctrl+Z  # 撤销
Ctrl+Y  # 恢复
Ctrl+]  # 缩进
Ctrl+[  # 减少缩进
Ctrl+F  # 查找，Enter找下一个，Shift+Enter，上一个
Ctrl+H  # 替换
Ctrl+K Ctrl+C # 注释
Ctrl+K Ctrl+U # 注释
Ctrl+/ # 切换注释

```
